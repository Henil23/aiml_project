## Pre-requisites:

- Anaconda installation
> Version: `conda 25.5.1`
- Conda environment under ./aiml_project. Run:
```cmd
conda create --prefix ./aiml_project python=3.11 ipykernel
```

> Tip: if you want to shorten the prompt name: 
```cmd 
conda config --set env_prompt (aiml_project) 
```
